SOBRE MI

¡Hola! Soy un apasionado desarrollador con experiencia en varios lenguajes y tecnologías web. Me considero un desarrollador full stack, con habilidades tanto en el frontend como en el backend. En el frontend, trabajo con JavaScript, HTML y CSS (incluyendo preprocesadores como SASS) para crear interfaces de usuario dinámicas y atractivas.

Además, tengo experiencia en el desarrollo backend utilizando PHP, lo que me proporciona una visión completa del desarrollo web y me permite abordar proyectos desde diferentes perspectivas.

Estoy emocionado de formar parte de proyectos que me permitan expandir mis habilidades y crear productos impactantes. ¡Espero poder trabajar contigo en el futuro!

Proyectos

Up Task

Up Task es una aplicación de gestión de tareas que te permite organizar y realizar un seguimiento de tus actividades diarias de manera eficiente.

Cuenta con funciones CRUD completas y una interfaz intuitiva, Up Task simplifica la gestión de tareas para ti y te ayuda a mantener un flujo de trabajo productivo.
App_Salon

App_Salon es una aplicación de agendación para barberías, diseñada para simplificar el proceso de reserva tanto para los clientes como para los administradores.

La aplicación permite a los clientes seleccionar fácilmente los servicios deseados y reservar citas en línea, mientras que los administradores pueden gestionar y visualizar todas las citas programadas.
